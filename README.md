## GitHub user search

Simple app for searching users from GitHub using their api

Tech:
- React
- Redux
- Redux saga
- ImmutableJS
- Webpack

Testing for all files with mocha / enzyme.

## Getting started
1. `npm install`
2. `npm start`

### Testing
run once:
`npm run test`
watch changes:
`npm run test:watch`

### Build

npm run build
builds to ./build
